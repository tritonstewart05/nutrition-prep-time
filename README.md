# Fitness-oriented Recipes and Their Correlation with Preparation Time

## Introduction

A common belief is that eating healthy, especially high-protein meals, takes more time than cooking regular meals. Because of this, many people avoid meal prepping since they think it is too time-consuming or inconvenient.

This project looks at whether that belief is actually true.

We will explore whether meals that are higher in protein and more nutrition-focused actually take longer to prepare than other types of meals. This matters because time is one of the biggest barriers people mention when it comes to eating healthier or getting in shape. If healthy meals do not take much longer to make, it challenges the idea that meal prepping is too time-intensive and suggests that eating well may be more realistic than people think.


## Dataset Overview

The dataset used in this project contains:

- **228,156 total recipes (rows)**

To answer our research question, we focus on the following relevant columns:

| Column | Description |
|--------|-------------|
| minutes | The total time required to prepare the recipe |
| tags | Labels that describe the type of dish (e.g., dinner, dessert, low-fat, easy, etc.) |
| protein_pdv | Protein content as a percentage of daily value |
| sat_fat_pdv | Saturated fat content as a percentage of daily value |
| calories | Total calories per recipe |
| n_steps | Number of preparation steps required |
